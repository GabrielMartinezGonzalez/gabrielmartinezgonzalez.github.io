    Definición de monomorfismo

    Una función es monomorfismo si es un homomorfismo inyectivo

      Notación asignación entre el dominio y el contradominio

      f:A↪Bf:A\hookrightarrow Bf:A↪B﻿

    Definición de epimorfismo

    Una función es epimorfismo si es un homomorfismo suprayectivo

      Notación asignación entre el dominio y el contradominio

      f:A↣Bf:A\rightarrowtail Bf:A↣B﻿

    Definición de isomorfismo

    Una función es isomorfismo si es un homomorfismo biyectivo

      Notación asignación entre el dominio y el contradominio

      f:A→~Bf:A\tilde\rightarrow Bf:A→~B﻿

    Definición de endomorfismo

    Una función es endomorfismo si es un homomorfismo cuyo dominio y contradominio coinciden

      Conjunto de endoformismos

      End(A)={f:A→A∣fEnd(A)=\{f:A\rightarrow A|fEnd(A)={f:A→A∣f﻿ es endomorfismo}\}}﻿

    Definición de automorfismo

    Una función es automorfismo si es un isomorfismo cuyo dominio y contradominio coinciden

      Conjunto de automorfismos

      Aut(A)={f:A→A∣fAut(A)=\{f:A\rightarrow A|fAut(A)={f:A→A∣f﻿ es automorfismo}\}}﻿
